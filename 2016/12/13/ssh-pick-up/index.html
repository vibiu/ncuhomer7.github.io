SSH Pick Up

发表于 2016-12-13 |

SSH 简介

SSH 几乎是每个 Linux 从业者必会技能，它能帮助我们快速远程连接终端，这里介绍下 SSH 的基础使用方法。

root用户登录

ssh -p 26661 root@45.78.27.220

vibiu用户登录

ssh -p 26661 vibiu@45.78.27.220

新建用户

groupadd admin  //用户组admin
useradd -d /home/vibiu -s /bin/bash -m bill  //用户vibiu
passwd vibiu  // 设置密码
usermod -a -G admin vibiu  //将vibiu添加到admin

设置sudo权限

visudo
打开/etc/sudoers
after
root   ALL=(ALL:ALL) ALL
vibiu  ALl=(ALL:ALL) ALL
vibiu  ALl=(ALL) NOPASSWD: ALL

ssh设置

authorized_keys文件的拷贝

// 本机端
cat ~/.ssh/id_rsa.pub | ssh vibiu@45.78.27.220 'mkdir -p .ssh && cat - >> ~/.ssh/authorized_keys'
// 或在服务器端
echo "ssh-rsa [your public key]" > ~/.ssh/authorized_keys

编辑etc/ssh/sshd_config

sudo cp /etc/ssh/sshd_config ~ //备份
sudo nnano /etc/ssh/sshd_config

端口修改 /etc/ssh/sshd_config

Port 26661

允许登陆的用户

AllowUser vibiu

改变authorized_keys文件的权限

sudo chmod 600 ~/.ssh/authorized_key && chmod 700 ~/.ssh/

在本机~/.ssh目录下创建cofig文件

Host ss
HostName 45.78.27.220
User vibiu
Port 25000

检查服务器区域设置

locale
//全部设成 en_US.UTF-8

visit Linux服务器的初步配置流程

保持连续链接

// open in client
sudo vim /etc/ssh/ssh_config
// add in
ServerAliveInterval 20
ServerAliveCountMax 888

// or open in server
sudo vim /etc/ssh/ssh_config
// add in
ClientAliveInterval 30
ClientAliveCountMax 6

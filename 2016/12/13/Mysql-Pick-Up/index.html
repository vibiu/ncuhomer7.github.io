mysql的配置

Install Mysql:

sudo apt-get install mysql-server

Install in centos

// 添加源
rpm -Uvh http://dev.mysql.com/get/mysql-community-release-e17-5.noarch.rpm
// 查看当前可用的mysql安装资源
yum repolist enabled | grep "mysql.*-community.*"
// 使用yum方式安装mysql(5.6)
yum -y install mysql-community-server
// 加入开机启动
systemctl enable mysqld
// 启动服务进程
systemctl start mysqld
//设置密码
mysql_secure_installation

see 阿里云CentOS 7.1使用yum安装MySql5.6.24

Root Login

mysql -u root-p

Survive in MySQL

\h      List of all Mysql commands

see How to Install MySQL on Ubuntu 14.04

新建用户

//sign as root
~$ mysql -u root -p
~$ password
//create user
mysql> insert into mysql.user(Host,User,Password) values("localhost","pythonadmin",password("123456"));
//refresh
flush privileges;
//a new user is created and he can sign as:
~$ mysql -u pythonadmin -p
~$ password

为用户授权

in root condition
//create a db for user
mysql> create database pythonDB;
//grant
mysql> grant all privileges on pythonDB.* to pythonadmin@localhost identified by "123456";
//refresh
mysql> flush privileges;
//part of privileges:
mysql> grant select update on pythonDB.* to pythonadmin@localhost identified by "123456";

see 如何给MySql创建连接用户并授权

删除用户

in root condition
mysql> delete from user where User="pythonadmin" and Host="localhost";
mysql> flush privileges;

修改用户密码

in root condition
mysql> updata mysql.user set password="654321" where User="pythonadmin" and Host="localhost";
mysql> flush privileges;

see MySQL新建用户

以特定的编码创建database

GBK: create database test2 default character set gbk collate gbk_chinease_ci;
UTF8: create database test2 default character set utf8 collate utf8_general_ci;

see MySQL创建database直接指定编码和排列

远程连接

mysql -u {{ usrname }} -h {{ 121.42.195.83 }} -P 3306 -D mydb -p

see 命令行如何远程连接MySQL数据库

安装mysqldb

pip install mysql-python
报错:mysql_config not found
解决:
1.whereis mysql_config
2.sudo apt-get install libmysqlclient-dev
3.将site.cfg中的mysql_config=/usr/local/mysql/bin/mysql_config这一行的#去掉,并且把mysql_config的路径设置正确

mysql && cpp

//安装依赖获得mysql.h 和 mysql.lib
// 这个过程会安装 mysql_config
sudo apt-get install libmysqlclient-dev

//gcc编译
gcc -o test test.c `mysql_config --cflags --libs`
